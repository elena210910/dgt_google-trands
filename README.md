# Descripción del proyecto
## Análisis de las matriculaciones de vehículos y su correlación con Google Trends
Este proyecto tiene como objetivo analizar los datos de las matriculaciones diarias de vehículos 
([Microdatos de Matriculaciones de Vehículos (diarios)](https://www.dgt.es/menusecundario/dgt-en-cifras/dgt-en-cifras-resultados/dgt-en-cifras-detalle/Microdatos-de-Matriculaciones-de-Vehiculos-diarios/), que incluyen información sobre los vehículos que se incorporan 
al Registro de Vehículos junto con sus características técnicas. Es razonable suponer que estos datos corresponden a vehículos nuevos, ya que son registrados oficialmente.
